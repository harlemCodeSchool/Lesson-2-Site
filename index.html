Javascript and the DOM

Now that we've rolling with HTML and CSS, it's time to start using javascript. Possibly the most powerful feature of js is it's ability to manipulate HTML and CSS.

Targeting Elements

Let's target an element that already exists. Right below this print is a div element with an id of 'jsTarget'. We will leave it blank with HTML and CSS and use js to control that stuff!

You can copy paste all this code into your script.js file anywhere inside the curly braces of the main function.

Turn '#jsTarget' into a rectangle with js:

var targetMyDiv = document.getElementById('jsTarget');
var styleDiv = targetMyDiv.style;

styleDiv.width = ('200px');
styleDiv.height = ('100px');
styleDiv.border = ('3px solid red');
styleDiv.backgroundColor = ('forestgreen');

Code breakdown

<<< Target element on the document w/id of 'jsTarget'

<<< Make styleDiv equal to calling the style function on

'jsTarget', which we targeted with the variable targetMyDiv

<<< Now set the width to 300px

<<< Now set the height to 200px

<<< Make the border solid, 2px and red

<<< Background color of forest-green
